# Stuff Features

This document outlines the main features of [Stuff](https://github.com/yousifsabah0/stuff).

- Simple authentication flow, password-less authentication with magic links
- Task management
  - Only five tasks per-day, with nested tasks
  - Flags
  - Brief description of the main task
  - Time based, e.g, morning and evening
  - Weekly and monthly planing
  - Projects
    - You can break each project into smaller parts
    - Assign priority and flags
  - Reminders
  - More to be discussed
- Daily and weekly habits (routines) tracking
  - Ability to flag a habit as something you're doing, or something you're quitting
- Achievement system
- Widgets
- TUI application
